1. SINK EQUATION (per D2, no amendment)
--------------------------------------------------------------------------------
   s(z) = n_S8/n_sat - 1;  L_S8(z) = C(z) * n_S8 * n_sat * h_w(s)  [cm^-3 s^-1]
   C(z) = rainout_rate_scale * Dg(z) * m_S8 / (rho_p * r_p^2)      [cm^3 s^-1]
   with m_S8 = 360.152/Navo g (conden.GAS_MASS_G_PER_MOL, master-verbatim),
   Dg from conden.build_conden_profile, and h_w the one-sided C1 hinge of D2.
   Verified pointwise limit: w -> 0 gives L -> C*n*(n - n_sat)_+, exactly the
   legacy gas-side loss branch (update_conden_rates k_pos applied as a
   first-order rate on n_S8). Signed/odd switches remain prohibited. The
   discrete rainout flux carries the stoichiometric factor 8:
   Phi_S,rain = 8 * sum_z L_S8(z)*dz(z)  [S atoms cm^-2 s^-1].
   In smooth_rainout mode the network's S8 <-> S8_l_s kinetics rows are inert
   (both rows zero; conden_branch, the window, and _activate_fix_species are
   skipped entirely); S8_l_s stays frozen at its initial ~0 (no reservoir).

2. SMOOTHING FUNCTION, WIDTH, SAFE ARITHMETIC
--------------------------------------------------------------------------------
   Canonical spec: dimensionless h_w(s), w ~ 0.1 (conden_smooth_width, a G5a
   ladder axis). Implementation: the algebraically identical division-free
   density form L = C * n * Phi_eps(x), x = n - n_sat, eps = w * n_sat_f,
   n_sat_f = max(n_sat, N_SAT_FLOOR) with N_SAT_FLOOR a named module constant
   (pure numerical floor, provenance-commented). Phi_eps(x) = 0 (x<=0);
   x^2/(2*eps) (0<x<eps); x - eps/2 (x>=eps). Every jnp.where branch is
   finite for ALL inputs (no NaN-through-where). dL/dn = C*(Phi + n*Phi')
   with Phi' in {0, x/eps, 1}: C1, enters the block-diagonal Jacobian
   analytically (gas row, local per layer; D5).
   Unit tests: analytic dL vs FD at finite w THROUGH ALL INPUTS (n, n_sat(T),
   C(T,P,Dg), boundary composition), guards exercised at n_sat extremes,
   pointwise w->0 kernel limit only (never a full solve near w=0).

   Budget (D6): dN_S,atm/dt = Phi_S,bottom - Phi_S,top - Phi_S,rain,
   Phi_S,rain = 8 * sum_z L_S8*dz.
   Measured body_fn order (outer_loop._make_runner): Ros2 solve -> clip ->
   atom_loss -> accept gate -> [conden branch: skipped in this mode] -> atm
   refresh -> hydrostatic renorm -> bottom pins -> commit.
   PER-OPERATOR, PER-ELEMENT discrete ledger (ALL elements, not only S/H):
   the exact inventory delta dN_E of (1) chemistry+transport+rainout+clip
   (the Ros2 step), (2) hydrostatic/abundance renormalization, (3) lower-
   boundary enforcement, (4) upper-boundary enforcement (asserted no-op),
   measured — never assumed (the pin is NOT assumed to change only two H
   and one S; renorm interactions are measured, not modeled). The terms
   telescope exactly to N_E(y_next) - N_E(y_prev).
   CLOSURE REQUIREMENT per open element E in {S, H}:
     dN_E - dN_E,boundary + dN_E,rain(E) ~ 0   (rain term zero for E != S)
   within a documented drift floor; the chemistry+transport ledger row for
   H must stay AT the closed-element drift floor — this REPLACES the old
   H-drift check with a sharper one rather than dropping it.
   ORDERING REQUIREMENT (review point 5): the closed-budget accept-gate and
